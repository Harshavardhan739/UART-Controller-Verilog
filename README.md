UART Controller Design in Verilog

Overview

This project implements a UART (Universal Asynchronous Receiver Transmitter) Controller using Verilog HDL. The design includes UART Transmitter, UART Receiver, Baud Rate Generator, and Verification Testbench.

Features

- UART Transmitter
- UART Receiver
- Baud Rate Generator
- Functional Verification
- Verilog HDL Design
